# Visão geral do monorepo

- `packages/` contém os pacotes publicáveis (e módulos internos)
- `example/` é uma aplicação Nest para testar integração

## Fluxos mais comuns

- Desenvolvimento com build watch: veja `pnpm run watch:packages` e `example/start:dev`
- Publicação: veja a página **Publicação**
